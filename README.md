# Calculator App
This calculator app is not like any ordinary calculator... It can
* Do basic calculations
* Recognize multiplication without the multiplication sign
* Repeat calculations
* Support both wrapping current input with square root or adding it to the end
* Storing values in variables that can be used anywhere in the expression (including the answer)
* Swiping the keyboard to type in equations
* Editing the expression at any point
* Allow users to have an interactive calculator widget on their phone
* Verbally ask for expressions to be solved by pressing a quick setting tool
